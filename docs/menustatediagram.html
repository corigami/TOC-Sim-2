Menu Activity Diagram

This diagram represents the menu state diagram for the Theory of Constraints Simulator 2.0 (TOC-Sim 2.0). It is utilizes the GoJS graphing framework.

The data to define the model is stored in the "/js/meni_state_model.js" file. Changes can be made to the model, click save and then paste the code into the state_model.js file.
